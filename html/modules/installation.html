üîß Troubleshooting Guide

‚ùå Installation Fails

Solution: Use --verbose flag for detailed output

./claude-installer.sh --full --verbose
# Check /tmp/claude-install.log for errors
# Retry with --force-reinstall if needed
üê≥ Docker Issues

Solution: Check Docker service and permissions

sudo systemctl start docker
sudo usermod -aG docker $USER
# Logout and login again
docker ps # Test Docker access
üîí Permission Errors

Solution: Use permission bypass or adjust permissions

claude --safe /task "your task"
# Or disable globally:
export CLAUDE_PERMISSION_BYPASS=false
claude /task "your task"
ü§ñ Agent Not Found

Solution: Refresh agent registry

claude --register-agents
claude --agents # List all available agents
claude --fix # Auto-detect and fix issues
üìä Performance Issues

Solution: Check hardware optimization settings

claude --status # Check system status
# Enable hardware optimization:
export ENABLE_AVX512=true
export METEOR_LAKE_OPTIMIZATION=true
üóÑÔ∏è Database Connection

Solution: Verify PostgreSQL container status

docker ps | grep claude-postgres
docker logs claude-postgres
# Restart if needed:
docker restart claude-postgres
