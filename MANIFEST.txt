SYSTEM REQUIREMENTS:
--------------------
- OS: Ubuntu 24.04+ LiveCD
- CPU: x86_64 (optimized for Intel Core Ultra)
- RAM: 2GB minimum
- Network: Internet connection required
- Disk: 100MB free space

FEATURES:
---------
✓ Claude Code (not CLI) installation
✓ Zero external dependencies - auto-installs npm/node
✓ LiveCD optimized (non-persistent)
✓ Auto permission bypass for Claude Code
✓ 28 production agents with local detection
✓ God-tier statusline (Neovim/Vim/Nano/Shell)
✓ Nano as default editor with syntax highlighting
✓ AVX512/AVX2 SIMD acceleration
✓ P-core/E-core aware for Intel hybrid CPUs
✓ First-launch helper with /config and /terminal-setup
✓ Local agents folder detection (same directory)

INSTALLATION:
-------------
1. Make scripts executable:
   chmod +x claude-quick-launch-agents.sh
   chmod +x claude-livecd-unified-with-agents.sh

2. Quick install (recommended):
   ./claude-quick-launch-agents.sh

3. Or manual install with options:
   ./claude-livecd-unified-with-agents.sh

WHAT GETS INSTALLED:
--------------------
- Claude Code → ~/.local/bin/claude
- Claude Agents → ~/.local/share/claude/agents/
- GitHub CLI → ~/.local/bin/gh
- Nano editor with enhanced config
- Statusline for Neovim/Vim/Nano/Shell
- Three commands:
  * claude (with auto permission bypass)
  * claude-normal (standard mode)
  * claude-first-launch (guided setup)
