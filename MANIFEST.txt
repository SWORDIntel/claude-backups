INSTALLATION:
-------------
1. Unified Installer v4.0 (RECOMMENDED):
   chmod +x installers/claude-installer.sh
   ./installers/claude-installer.sh --full
   
   Options:
   --quick    # Minimal components
   --full     # All features (agents, orchestration, statusline)
   --portable # Self-contained installation
   --custom   # Choose components
   --auto     # No prompts
   --dry-run  # Test installation

2. Quick install (legacy):
   ./installers/claude-quick-launch-agents.sh

3. Portable install (self-contained):
   ./installers/claude-portable-launch.sh

4. Manual install with options:
   ./installers/claude-livecd-unified-with-agents.sh
