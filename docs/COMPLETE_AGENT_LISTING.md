## Agent Status Classification

### Active Agents (Fully Implemented)
- File size > 100 bytes
- Contains implementation sections
- Has proper YAML frontmatter or structured content
- Includes purpose, tools, and coordination patterns

### Template Agents (Basic Structure)
- Has basic structure but limited implementation
- May be missing key sections
- Typically used as starting points

### Stub Agents (Placeholders)
- File size < 100 bytes
- Minimal or no content
- Placeholder for future implementation
