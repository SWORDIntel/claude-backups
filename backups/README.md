# Consolidated Backup Directory

This directory contains all backup files for the Claude Agent Framework project.

## Structure

- **precision-orchestration_*.md** - Output style backups from various dates
- **claude-installer.sh.backup*** - Installer script backups
- **agents-backups/** - Agent-related backups and archives
- **optimizer-backups/** - Python script optimizer backups
- **backup-20250825-*/** - Learning system component backups
- **setup_learning_system.backup.py** - Learning system setup backup

## Backup Policy

All backups should be placed in this consolidated directory to maintain organization.
Subdirectories are used for categorizing different types of backups.

## Cleanup

Old backups can be safely removed after verifying the current system is stable.
Consider archiving important backups before deletion.
